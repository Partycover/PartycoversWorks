About Older Versions

Pokemon Rejuvenation is developed over the course of several versions. Development initially began with the private versions which were exclusive to the creator's, Jan's, friends, featuring a largely different plot. These are sometimes called the "negative versions", as Eizen, in his first appearance iterates through them before iterating through the public versions; versions 1 to 13. A copy of one of the private builds of Pokemon Rejuvenation can be found on Jan's patreon.

The public versions which are the only versions I will ever discuss (unless Jan releases the private versions), were initially developed alongside the private versions, roughly at the time of Version 7's development, the development of private versions's ceased and all effort was given to the public versions.

Each of Pokemon Rejuvenation's Versions don't just add new content, but they also often rewrite old content! This is often for the best, as old/weak plot points can be replaced or rewritten!

Why does this site exist?

Older Versions are interesting! Playing older versions of Pokemon Rejuvenation is really the best way to experience it, but there are tons of different versions, sometimes with content only exclusive to a certain range. For example, if you were to playthrough Version 9, you'd still never experience things such as Blacksteeple Prison (which would later become Blacksteeple Castle). This is sort of why I bother sharing stuff like this. I usually share most things regarding Older Versions on Reddit or YouTube but YouTube isn't the best for everything I want to share which aren't really long enough to make videos out of. This site is hopefuly going to be a "catch-all" for all things Older Versions.
